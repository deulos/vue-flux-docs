# Installation

Installation is very simple, just use your favourite package manager:

npm install --save vue-flux@stable

# Usage

In your code just add the component in the template.

<vue-flux
   :options="fluxOptions"
   :images="fluxImages"
   :transitions="fluxTransitions"
   ref="slider">
      <flux-pagination slot="pagination"></flux-pagination>
</vue-flux>

<button @click="$refs.slider.showImage('next')">NEXT</button>

And the data that will be used in the template.

import { VueFlux, FluxPagination, Transitions } from 'vue-flux';

export default {
   components: {
      VueFlux,
      FluxPagination
   },

   data: () => ({
      fluxOptions: {
         autoplay: true
      },
      fluxImages: [
         'URL1',
         'URL2',
         'URL3'
      ],
      fluxTransitions: {
         transitionBook: Transitions.transitionBook
      }
   })
}

If you want to see more details about configuration check the VueFlux component documentation.
